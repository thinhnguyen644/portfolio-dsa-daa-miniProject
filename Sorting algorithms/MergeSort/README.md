# 📘 MERGE SORT

Merge Sort là một thuật toán sắp xếp theo kiểu chia để trị (Divide and Conquer), được sử dụng rộng rãi vì độ ổn định và hiệu quả trong việc xử lý các tập dữ liệu lớn.
